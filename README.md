# PointiFlow
ポイントインカムの回収作業の一部自動化拡張機能  
## 毎日クリックの回収
[毎日クリックページ](https://pointi.jp/daily.php)  
自動でptゲットボタン押して回収します
## お楽しみアンケートの自動回答
[パーソナライズアンケート](https://pointi.jp/contents/research/personalize_enquete/)  
毎日配信されているお楽しみアンケートを開くと、自動で回答選択+応募券獲得コードの入力をしてポイントを回収します
## 読んで貯める系の自動回収
[アットプレス](https://pointi.jp/contents/magazine/atpress/)、[サンキュ](https://pointi.jp/contents/magazine/bene/)、[イチオシ](https://pointi.jp/contents/magazine/ichioshi/)等

それぞれのページを開くと記事開いてpt回収までを自動で行います。  
デフォルトでは2秒間隔で新しい記事開くようになってるので、速すぎる場合は自分で調整してください
